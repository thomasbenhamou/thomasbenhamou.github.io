A jobboard for local winter jobs

See link:

WinterJobs

 Be patient while Heroku loads up its dyno!

Made from scratch in and using:

  • Symfony 4
  • Doctrine 2 ORM
  • Bootstrap

Allow users to browse winter jobs in ski resorts.

Main functionnalities

  • User sign up and registration using Symfony's authentication and authorization tools: users have different roles, either normal user or admin user. The latter is able to manage all the job adverts (read, update, delete)
    • On sign up, password is hashed
  • If signed up, a user can post a new job advert
    • The form is validated using Doctrine and Symfony constraints on password, emails, etc.
    • The advert's address is geocoded using Google Maps JS API
  • On the main page, adverts are displayed along with a Google Maps showing custom markers and job infos
  • Once signed up, the user can:
    • See the adverts that he has already posted
    • Update and/or delete them
